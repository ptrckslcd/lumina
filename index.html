Browser Mixed Content Block Detected

Your app is served over HTTPS (for example on Vercel), but local WLED REST calls use HTTP. Most browsers block this by default as mixed content.

When to use this: Use this only when accessing Lumina through a hosted/public URL (such as Vercel). If you are running on localhost/local network, this step is usually not required.

Desktop Guide

  1. Click the View Site Information icon (lock or controls) beside the URL.
  2. Open Site Settings and locate the Insecure Content permission.
  3. Change it from Block to Allow, then refresh the page.

Mobile Guide

  1. Tap the browser site icon or menu near the address bar for site details.
  2. Open Site Settings (or Permissions) and find Insecure Content.
  3. Switch it to Allow, then reload the dashboard.
